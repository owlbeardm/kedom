                            # Leveling-Up Checklist

                            Note

                            If you are using character.pf2.tools some of the steps can be automated (Hit Point increase, proficiency bonuses calculation, etc. see below).

                            1. Increase your level by 1.
                              1. Increase your maximum Hit Points by the amount listed in your class entryopen in new window (done automatically in character.pf2.tools).
                            2. Add class features from your classopen in new window advancement table.
                              1. Select feats as indicated on your classopen in new window advancement table. For ancestry feats, see Ancestriesopen in new window. For class feats, see your class entryopen in new window. For general feats and skill feats, see Featsopen in new window (skill feats type of general feats, so you can select skill feat in place of general feat).
                              2. Add spells and spell slots if your class grants spellcasting. See Spellsopen in new window for spells. If your class has signature spells feature select them. (Note that your cantrips and focus spells heightened to maximum spell level available to you)
                              3. Add other features (like skill increase or expertises).
                            3. Gain ability boosts as staited in Gradual Ability Boostsopen in new window (this is a varian rule).
                            4. Add new bonus from Automatic Bonus Progressionopen in new window table (this is a varian rule).
                            5. Adjust bonuses from feats and other abilities that are based on your level or you proficiencies.

                            # Leveling-Up using character.pf2.tools

                            # 1. Level increase

                            # 2. Class features

                            Add class features from your classopen in new window advancement table. Proficiencies can be changed for everething if corresponding menu. For Merlin it would be general feat and skill increase (Thorough Search and Arcana).

                            Note

                            Spellcasting and skill have different proficiency. Occultism and occult spellcasting for example.

                            # 3. Ability Boost

                            Gain ability boosts as staited in Gradual Ability Boostsopen in new window.

                            # 4. Automatic Bonus Progression

                            Add new bonus from Automatic Bonus Progressionopen in new window table.

                            # 5. Other

                            If needed adjust bonuses from feats and other abilities that are based on your level or you proficiencies. (New formulas, extra damage, new familiar abilities...)

                            # Finished character

                            Finised Merlin's sheetopen in new window.

                            *[class features]: Abilities that you gain from you class. Can be found in table on class page.
                            *[skill feat]: At 2nd level and every 2 levels thereafter, most classes gain a skill feat, though rogues gain them earlier and more often. Your character must be trained in the corresponding skill to take a skill feat.
                            *[general feat]: Most classes grant a general feat at 3rd level and every 4 levels thereafter. At each of these levels, you can select any general feat (including skill feats) as long as your character qualifies for it.
                            *[ability boosts]: Boost for your ability score. An ability boost normally increases an ability score‚Äôs value by 2.
